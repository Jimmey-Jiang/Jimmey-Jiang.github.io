TABLE_SCHEMA=@schema_name and table_name=p_tablename and index_name=p_idxname; if @cnt > 0 then call excuteproc(@str); end if; end ; ``` 7,对于大型的数据我们直接通过脚本直接操作数据库，不必通过底层orm框架。当然也可以abp在原有的基础上扩展
